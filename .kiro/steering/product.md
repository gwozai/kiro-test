# Product Overview

This repository contains crawled web data from various websites. It serves as a data storage for web scraping results.

## Purpose
- Store structured web crawl results in JSON format
- Preserve article metadata (titles, URLs, timestamps)
- Maintain content previews from crawled pages

## Data Characteristics
- Each page file represents a single crawled web page
- Data includes pagination information from source sites
- Articles are extracted with title and URL pairs
- Content previews capture raw text from pages
